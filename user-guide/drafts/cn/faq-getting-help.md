# 获取帮助

> 哪里找答案、如何提问、如何提交问题

---

## 先自助

- 📖 **用户指南**：当前文档集（安装、场景、FAQ）。  
- 🔍 **GitHub Issues 搜索**：https://github.com/Testany-io/srs-writer-plugin/issues  
  - 同时查看已关闭的 issue  
  - 使用错误关键词搜索（如 “No AI model available”）

---

## 社区提问（GitHub Discussions）

- 链接：https://github.com/Testany-io/srs-writer-plugin/discussions  
- 版块建议：`Q&A` 提问，`Show and Tell` 分享经验，`Ideas` 提功能建议  
- 提问要点：
  1. 简述目标/遇到的问题  
  2. 提供环境：VS Code 版本、SRS Writer 版本（0.8.2）、操作系统、是否登录 Copilot  
  3. 给出重现步骤或示例指令/文件路径  
  4. 粘贴关键报错（聊天面板或开发者工具 Console）

---

## 报告 Bug（GitHub Issues）

- 创建 Issue：https://github.com/Testany-io/srs-writer-plugin/issues/new/choose  
- 请包含：
  - 环境：VS Code 版本、SRS Writer 版本 0.8.2、OS、Copilot 是否登录  
  - 重现步骤（越短越好）  
  - 预期结果 vs 实际结果  
  - 完整错误信息/日志（`帮助` → `切换开发人员工具` → Console）

---

## 需要快速定位问题？

- 先运行控制面板的 **同步状态检查**；若仍异常尝试 **Force Sync Context**。  
- 复制错误文本在 Issues/Discussions 中搜索；通常已有解决方案。  
- 无法自行解决时，在 Issue 中附上步骤、日志和版本信息。

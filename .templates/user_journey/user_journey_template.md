## User Journeys

### 用户角色定义
#### 主要用户 - 普通用户
- **背景**: 需要使用系统完成日常工作的员工
- **技能水平**: 中等计算机操作技能
- **目标**: 高效完成工作任务，减少操作复杂度
- **痛点**: 系统复杂、响应慢、操作繁琐
